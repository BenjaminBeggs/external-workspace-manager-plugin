Used as root path to compute the workspace path on this disk. Optional. If provided, it must be a relative path.

The workspace path on disk is computed based on the following formula: physicalPathOnDisk/$JOB_NAME/$BUILD_NUMBER

When the plugin allocates the workspace path to be used within a Node, then the complete workspace path has the next pattern, starting with the mounting point from the Jenkins Node to the actual disk: /mountPointFromNodeToDisk/physicalPathOnDisk/$JOB_NAME/$BUILD_NUMBER
