If you have multiple Nodes that share the same label, you can make use of this template to provide a common External Workspace config for all the nodes. Instead of specifying in each Node config the Disk ID and the Local root path parameters, they can be provided globally in the External Workspace Templates.

Note: If you specify the Disk ID and the Local root path in the External Workspace Templates for a specific Node label, then these parameters will be taken into consideration by the plugin. If the parameters are also provided in a Node config having the given label, then the parameters from the Node config will be ignored. But, if these two parameters are not provided in the External Workspace Templates, they must be mandatory provided in each Node config.
